Project Overview

The Digital Library System is a comprehensive full-stack application that manages media collections including books, films, and magazines. Built with Python, it features a Flask REST API backend and a Tkinter GUI frontend.

  • RESTful API with Flask
  • Tkinter Desktop Interface
  • JSON-based Data Storage
  • CRUD Operations
  • Search & Filter Capabilities

API Endpoints

GET /media # List all media items
GET /media/{id} # Get specific media
POST /media # Add new media
PUT /media/{id} # Update media
DELETE /media/{id} # Delete media
GET /media/category/{type} # Filter by category

Sample Media Data Structure

{ "id": 1, "title": "The Great Gatsby", "type": "book", "author": "F. Scott Fitzgerald", "year": 1925, "category": "Fiction", "description": "A classic novel about the American Dream" }
